


**校园乐享圈**

**2022年1月 - 2022年6月**

**项目背景:**

校园乐享圈是一个专为大学生打造的校园生活服务平台。设有博客分享功能，用户可以通过签到、点赞、关注等互动方式，与好友分享自己的校园生活

**技术描述:**

1. 负责开发一人一单功能,**对业务进行技术迭最终使用分布式锁解决一人一单问题**
    
2. 使用 Redis Set 数据结构实现**好友关注、共同关注**、Timeline 类型的 **Feed 流模式的关注推送**
    
3. 负责商户查询功能,**解决 redis 与数据库双写一致、缓存穿透、缓存击穿问题,**解决高并发情况下的数据库压力过大的问题
    
4. 负责登录功能,**使用 redis+token解决session共享问题**,隐藏用户敏感信息,并**使用双重拦截器解决token刷新的问题**
    
5. 负责优惠卷**秒杀功能**,使用redis实现全局唯一id替代数据库自增id,使用**乐观锁解决高并发情况下库存超卖问题**,并使用RabbitMQ进行秒杀优化,实测90qps优化到500qps


**主要工作：**

6. **参与组内冷热分离技术讨论并最终落地，节约公司成本，让存储架构能支撑未来3到5年的业务发展。**
    
7. 注册功能开发：对加密算法技术选型，**使用Argon2算法进行加密加盐防止彩虹攻击**，使用ZCache+Token解决session共享问题。
    
8. 参与转账功能迭代：解决空回滚、防悬挂等问题，并解决大商户数据倾斜问题。
    
9. **设计分库分表路由策略，使用2次幂提升路由性能**并且以后新增分库分表数量对数据迁移也有一定优势。

10. 业务明细查询开发：对不携带分库分表字段情况下走全表的问题**使用二次查询法+多线程进行性能优化。**
    
11. **开发通用限流，使用AOP+注解+RatteLimited对部分接口限流，并支持根据ip限流。**
